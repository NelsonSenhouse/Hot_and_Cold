How to Play:

Guess the random number!

1. Add and subtract to modify guess (number is from 1-100 inclusive)

2. Press "commit" to enter guess

3. Your guess will be given feedback according to the following criteria:

  • Very Hot: Guess is within 5
  • Hot: Guess is within 6-8
  • Very Warm: Guess is within 9-15
  • Warm: Guess is within 16-20
  • Cool: Guess is within 21-30
  • Very Cool: Guess is within 31-40
  • Cold: Guess is within 41-55
  • Very Cold: Guess is over 55 away

4. Each commit (except for "Very Hot") will use up a "guess". If you run out of guesses, you lose!

5. Guess the number correct and you win!
